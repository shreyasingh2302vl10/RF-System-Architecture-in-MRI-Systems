# RF-System-Architecture-in-MRI-Systems

##  Summary

###  Core Principle
- Hydrogen spins in magnetic field B0  
- Rotate at Larmor frequency  

f = γ × B0

---

###  RF Excitation
- RF coil generates B1 ⟂ B0  
- Spins get tilted (flip angle θ)  

Resonance condition:
f_RF = f_Larmor

---

###  Signal Generation
- After RF OFF → spins relax & emit RF signal  
- Induced voltage in coil = MRI signal  

---

###  RF Receiver Chain
- Weak signal → amplified using LNA  
- Frequency conversion using mixer
